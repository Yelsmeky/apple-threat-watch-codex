# Contributing

This repository is intentionally maintained as a small, prompt-only reference.

To keep the trust model simple:

- Pull requests are not expected.
- Executable code is not accepted.
- LaunchAgent plist templates are not accepted.
- Malware samples, exploit code, binaries, archives, packages, encoded payloads, or installer scripts are not accepted.

If you have feedback, please open an issue instead.

Useful issue reports include:

- unclear installation instructions
- safety-model questions
- source-quality suggestions
- wording improvements
- bug reports about the prompt-only installation flow

The maintainer may close issues that request executable code, malware analysis tooling, or active remediation features.
